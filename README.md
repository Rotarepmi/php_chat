# php_chat
### Simple public chat application (PHP, Ajax)
```
You can register and login to write in facebooklike chat window to other users.
This chat was built using PHP.
```
### Live: [mandra.pl - chat](http://mandra.pl/z3/index.php)
### Stack:
* PHP
* JS (JQuery)
* Sass
* HTML
* Bootstrap 4
